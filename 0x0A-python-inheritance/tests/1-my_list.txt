The ``1-my_list`` module
============================

Using ``print_sorted``
-----------------------

Now use it:

    >>> my_list = MyList()

    >>> my_list.print_sorted()
    []

    >>> my_list.append(1)

    >>> my_list.append(4)

    >>> my_list.append(2)

    >>> my_list.append(3)

    >>> my_list.append(5)

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    >>> my_list.append(-5)

    >>> my_list.print_sorted()
    [-5, 1, 2, 3, 4, 5]
